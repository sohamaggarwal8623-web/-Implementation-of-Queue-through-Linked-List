# Queue Implementation Using Linked List in C

This project implements a **Queue Data Structure using Linked List** in the C programming language.

## Features
- Enqueue operation
- Dequeue operation
- Queue traversal/printing
- Empty queue checking
- Dynamic memory allocation using `malloc`
- Memory deallocation using `free`

## Concepts Used
- Structures in C
- Pointers
- Dynamic Memory Allocation
- Linked Lists
- Queue Data Structure (FIFO)

## Queue Operations

### Enqueue
Adds an element to the rear of the queue.

### Dequeue
Removes an element from the front of the queue.

### Print
Displays all elements currently present in the queue.

## Code Example

```c
enqueue(q,10);
enqueue(q,20);
enqueue(q,30);

dequeue(q);

print(q);
